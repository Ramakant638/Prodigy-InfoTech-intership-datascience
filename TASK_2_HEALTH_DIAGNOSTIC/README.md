# TASK 2: HEALTH DIAGNOSTIC

## Project Name
Health Diagnostic Analysis using Python

## Internship Domain
Data Science

## Project Description
This project analyzes patient health data using Python.  
It uses a manually created health database and classifies patients into Healthy, Moderate Risk, or High Risk categories based on blood pressure, sugar level, cholesterol, and heart rate.

## Technologies Used
- Python
- Pandas
- Matplotlib

## Dataset
The dataset is created manually inside the Python file.

The dataset contains:
- Patient ID
- Age
- Gender
- Blood Pressure
- Sugar Level
- Cholesterol
- Heart Rate

## Features
- Creates patient health diagnostic database
- Checks blood pressure level
- Checks sugar level
- Checks cholesterol level
- Checks heart rate
- Assigns risk score
- Classifies patients into health risk categories
- Finds high-risk patients
- Calculates average health values
- Shows health diagnostic graphs

## Diagnosis Categories
- Healthy
- Moderate Risk
- High Risk

## Algorithm
1. Start the program
2. Import Pandas and Matplotlib
3. Create health diagnostic database
4. Convert database into DataFrame
5. Define diagnosis function
6. Check blood pressure, sugar level, cholesterol, and heart rate
7. Assign risk score to each patient
8. Classify patient as Healthy, Moderate Risk, or High Risk
9. Display final diagnosis result
10. Find high-risk patients
11. Calculate average health values
12. Plot graphs for analysis
13. Print final conclusion
14. End the program

## How to Run
Install required libraries:

```bash
pip install pandas matplotlib
```

Run the Python file:

```bash
python health_diagnostic_analysis.py
```

## Output
The project displays:
- Health diagnostic database
- Diagnosis category summary
- High-risk patients
- Average blood pressure
- Average sugar level
- Average cholesterol
- Average heart rate
- Diagnosis graph
- Blood pressure graph
- Sugar level graph
- Cholesterol graph

## Conclusion
This project shows how Data Science can be used to analyze health data and support basic diagnostic decision-making.
