# TASK 3: E-COMMERCE ANALYSIS

## Project Name
E-Commerce Sales Analysis using Python

## Internship Domain
Data Science

## Project Description
This project analyzes e-commerce sales data using Python.  
It uses a manually created e-commerce database and performs sales analysis, product analysis, category analysis, payment mode analysis, and business insight generation.

## Technologies Used
- Python
- Pandas
- Matplotlib

## Dataset
The dataset is created manually inside the Python file.

The dataset contains:
- Order ID
- Customer Name
- Product
- Category
- Quantity
- Price
- Payment Mode

## Features
- Creates e-commerce sales database
- Calculates total amount for each order
- Calculates total sales
- Performs product-wise sales analysis
- Performs category-wise sales analysis
- Analyzes payment modes
- Finds best-selling product
- Finds highest order amount
- Shows sales analysis graphs

## Algorithm
1. Start the program
2. Import Pandas and Matplotlib
3. Create e-commerce sales database
4. Convert database into DataFrame
5. Calculate total amount using Quantity × Price
6. Display complete sales database
7. Calculate total sales amount
8. Analyze product-wise sales
9. Analyze category-wise sales
10. Count payment modes
11. Find best-selling product
12. Find highest order amount
13. Plot category-wise sales graph
14. Plot product-wise sales graph
15. Plot payment mode graph
16. Plot quantity sold graph
17. Print final conclusion
18. End the program

## How to Run
Install required libraries:

```bash
pip install pandas matplotlib
```

Run the Python file:

```bash
python ecommerce_sales_analysis.py
```

## Output
The project displays:
- E-commerce sales database
- Total sales amount
- Product-wise sales
- Category-wise sales
- Payment mode count
- Best-selling product
- Highest order details
- Category-wise sales graph
- Product-wise sales graph
- Payment mode graph
- Quantity sold graph

## Conclusion
This project shows how Data Science can be used to analyze e-commerce sales data and generate useful business insights.
