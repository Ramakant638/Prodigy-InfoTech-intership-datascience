# TASK 4: EDUCATION ANALYSIS

## Project Name
Student Performance Analysis using Python

## Internship Domain
Data Science

## Project Description
This project analyzes student academic performance using Python. It uses a manually created education database and performs marks analysis, percentage calculation, grade classification, attendance analysis, and student performance visualization.

## Technologies Used
- Python
- Pandas
- Matplotlib

## Dataset
The dataset is created manually inside the Python file.

The dataset contains:
- Student ID
- Student Name
- Class
- Maths Marks
- Science Marks
- English Marks
- Attendance

## Features
- Creates student performance database
- Calculates total marks
- Calculates percentage
- Assigns grades
- Finds topper
- Finds lowest performer
- Calculates subject-wise average marks
- Analyzes attendance
- Shows student performance graphs

## Algorithm
1. Start the program
2. Import Pandas and Matplotlib
3. Create student performance database
4. Convert database into DataFrame
5. Calculate total marks
6. Calculate percentage
7. Define grade function
8. Assign grade to each student
9. Display final student result
10. Find topper
11. Find lowest performer
12. Calculate subject-wise averages
13. Count grades
14. Plot percentage graph
15. Plot subject average graph
16. Plot attendance graph
17. Plot grade distribution graph
18. Print final conclusion
19. End the program

## How to Run
Install required libraries:

```bash
pip install pandas matplotlib
```

Run the Python file:

```bash
python education_analysis.py
```

## Output
The project displays student result, total marks, percentage, grade, topper details, lowest performer details, subject average marks, grade summary, and graphs.

## Conclusion
This project shows how Data Science can be used to analyze educational data and understand student performance.
