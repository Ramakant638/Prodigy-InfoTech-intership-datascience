# TASK 1: STOCK ANALYSIS

## Project Name
TCS Stock Market Analysis using Python

## Internship Domain
Data Science

## Project Description
This project analyzes Tata Consultancy Services (TCS) stock market data using Python.  
It uses a manually created stock database and performs stock trend analysis, moving average calculation, trading volume analysis, and bullish or bearish trend identification.

## Technologies Used
- Python
- Pandas
- Matplotlib

## Dataset
The dataset is created manually inside the Python file.

The dataset contains:
- Date
- Open Price
- High Price
- Low Price
- Close Price
- Volume

## Features
- Creates TCS stock market database
- Calculates daily return percentage
- Calculates 3-month moving average
- Finds highest stock price
- Finds lowest stock price
- Finds highest trading volume
- Shows closing price graph
- Shows moving average graph
- Shows trading volume graph
- Gives bullish or bearish market trend

## Algorithm
1. Start the program
2. Import Pandas and Matplotlib
3. Create TCS stock market database
4. Convert the database into DataFrame
5. Convert Date column into date format
6. Calculate daily return percentage
7. Calculate moving average
8. Find highest stock price
9. Find lowest stock price
10. Find highest trading volume
11. Compare first closing price and last closing price
12. Display bullish or bearish trend
13. Plot graphs for analysis
14. Print final conclusion
15. End the program

## How to Run
Install required libraries:

```bash
pip install pandas matplotlib
```

Run the Python file:

```bash
python tcs_stock_analysis.py
```

## Output
The project displays:
- TCS stock database
- Highest stock price
- Lowest stock price
- Highest trading volume
- Overall market trend
- Closing price graph
- Moving average graph
- Trading volume graph

## Conclusion
This project shows how Data Science can be used to analyze stock market data and understand stock price movement.
